# 📦 顺丰电商退货截图模拟器 (SF-Express Return Mockup)

这是一个基于 Vue 3 和 Tailwind CSS 开发的静态网页工具，旨在帮助电商从业者或开发者快速生成高度还原的顺丰退货界面截图。

### ✨ 功能特点
- **多平台适配**：支持一键切换 **拼多多、京东、淘宝/天猫** 三种主流电商退货视觉风格。
- **高度自定义**：可实时修改收件人、联系电话、退货地址及倒计时时间。
- **高清导出**：集成 `html2canvas`，支持 3 倍分辨率导出高清 PNG 图片。
- **纯前端实现**：无需服务器，支持 GitHub Pages 秒级部署。

### 🚀 快速开始
1. Fork 本仓库。
2. 进入仓库设置 `Settings` -> `Pages`。
3. 选择 `main` 分支并点击保存。
4. 访问你的专属 URL 即可开始使用。

### 🛠️ 技术栈
- Vue 3 (Composition API)
- Tailwind CSS (UI 布局)
- html2canvas (截图组件)
